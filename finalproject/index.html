Site Name: OyeWrites

The site name "OyeWrites" is a fusion of my name and my passion for writing code. It serves as a personal brand that is both unique and relevant to my profession as a web developer.

Site Purpose

The primary purpose of this website is to showcase my skills and completed projects as a web developer. It provides a platform for potential clients, employers, and collaborators to view my work, understand my technical abilities, and get in touch with me for new opportunities.

Scenarios

This site is built to answer key questions from visitors:


  • Scenario 1: Can I see examples of responsive websites you have built?
  • Scenario 2: How can I get in touch with you about a potential project?

Color Schema

The design utilizes a professional and modern color palette:


  • --primary-color: #000000 (Black) is used for backgrounds on hero sections and footers to create a strong contrast.
  • --secondary-color: #2c3e50 (Dark Slate Blue) is used for headings, primary buttons, texts, and navigation links on light backgrounds.
  • --accent-color: #e67e22 (Orange) is used to draw attention to call-to-action buttons, hover effects, and key information.
  • --light-bg: #f4f4f4 (Light Gray) is the main background color for body sections.

Typography

The entire website uses the "Poppins" font family. It is a clean, modern, and highly legible sans-serif font that enhances readability and gives the site a professional, approachable feel.
